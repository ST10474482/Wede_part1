# Belem Bakery Website

A simple, clean website for **Belem Bakery**, a beloved Portuguese bakery in Johannesburg, showcasing their story, menu, reviews, contact info, and ordering options. The website is fully responsive, SEO-friendly, and designed for optimal user experience.

---

# PART 1 – Overview

## Features

- **Home:** Welcome section and bakery highlights  
- **About:** History, mission, vision, and team  
- **Menu:** Photos and descriptions of popular baked goods  
- **Reviews:** Customer testimonials  
- **Contact:** Contact form, phone, email, and location map  
- **Order:** Ordering instructions with delivery app links  

## Project Structure

- `index.html` – Home page  
- `about.html` – About the bakery  
- `menu.html` – Menu with baked goods  
- `reviews.html` – Customer reviews  
- `contact.html` – Contact form and info  
- `order.html` – Ordering details  
- `images/` – Images for products and branding  
- `README.md` – This file

# PART 2 – Pages & Features

This part focuses on **improving structure, responsiveness, and consistency** across all pages using external CSS and mobile-friendly navigation.

## Pages Included

### 1. Home – `index.html`
- Introduces Belem Bakery with a welcoming hero section  
- Lists key highlights: freshness, tradition, dine-in, catering

### 2. About – `about.html`
- Shares the bakery’s history, mission, vision, and team  
- Semantic structure for easy readability and styling  

### 3. Menu – `menu.html`
- Displays bakery offerings in a responsive product grid

### 4. Contact – `contact.html`

- Simple and direct call-to-action section  

### 6. Reviews – `reviews.html`
- Customer testimonials presented in a grid format  
- Highlights authenticity and customer satisfaction  

## Features

# PART 3 – SEO, Performance & Interactivity

This part focuses on **SEO optimization, performance improvements, and enhanced interactivity**.

## SEO & Metadata

- Meta tags: `description`, `keywords`, `author` on all pages  
- `robots.txt` to control search engine crawling:  
  - Disallow private folders, JS, and CSS if needed  
  - Allow gallery images to be crawled  
- `sitemap.xml` with all main pages listed for better indexing  

## Interactivity Enhancements

- JavaScript-enabled hamburger menu for mobile navigation  
- Modal popup for order confirmation  
- Animate-on-scroll effects (AOS) on Reviews page  
- Lazy-loading images for faster page load  

## Pages Updates

### 1. Reviews – `reviews.html`
- Customer testimonials in animated grid  
- Smooth scroll and fade-in effects improve user engagement  

### 2. Order – `order.html`
- Order form with modal confirmation on submission  
- Fields validated with HTML5 `required` attributes  

### 3. Enquiry – `enquiry.html`
- Simple form for additional inquiries  
- Maintains responsive layout and consistent styling  

## Additional Features

- Fully responsive design across all pages  
- Semantic HTML5 for accessibility and SEO  
- External CSS and JS linked consistently for maintainability  
- Google Fonts included for typography consistency
