# PART 1

# Belem Bakery Website

A simple, clean website for Belem Bakery, a beloved Portuguese bakery in Johannesburg, showcasing their story, menu, reviews, contact info, and ordering options.

## Features

- Home: Welcome and bakery highlights  
- About: History, mission, vision, and team  
- Menu: Photos and descriptions of popular baked goods  
- Reviews: Customer testimonials  
- Contact: Contact form, phone, email, and location map  
- Order: Ordering instructions with delivery app links  


## Project Structure

- index.html # Home page
- about.html # About the bakery
- menu.html # Menu with baked goods
- reviews.html # Customer reviews
- contact.html # Contact form and info
- order.html # Ordering details
- images/ # Images for products and branding
- README.md # This file

# PART 2

# Belem Bakery Website

This is the second part of the Belem Bakery website project, focusing on improving structure, responsiveness, and consistency across all pages using external CSS and a mobile-friendly navigation menu.

## Pages Included

### 1. Home-index.html
- Introduces Belem Bakery with a welcoming hero section.
- Lists key highlights (freshness, tradition, dine-in, catering).
- Fully responsive with linked CSS and mobile toggle.

### 2. About- about.html
- Shares the bakery’s history, mission, vision, and team.
- Semantic structure for easy readability and styling.

### 3. Menu-menu.html
- Displays bakery offerings in a responsive product grid.
- Includes images, descriptions, and prices for each item.

### 4. Contact-contact.html
- Provides a contact form (name, email, message).
- Displays phone number, email, and physical address.
- Embedded Google Maps location.
- Responsive layout with external CSS.

### 5. Order Now-order.html
- Instructions for placing orders via phone, Uber Eats, or Mr D.
- Simple and direct call-to-action section.

### 6. Reviews-reviews.html
- Customer testimonials presented in a grid format.
- Highlights authenticity and customer satisfaction.

##  Features

-  External CSS (`css/style.css`) applied consistently  
-  Responsive mobile navigation using hamburger menu  
-  Semantic HTML5 structure across all pages  
-  Lazy loading for images  
-  Embedded Google Map on Contact page
